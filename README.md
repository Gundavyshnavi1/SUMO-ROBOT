# Bluetooth Controlled Sumo Robot

## Overview
The Bluetooth Controlled Sumo Robot is a compact robotic system designed to push the opponent out of the competition arena using powerful motors and efficient movement control. The robot is controlled wirelessly through a Bluetooth module and a mobile application, allowing the user to move the robot in different directions during a sumo robot competition.

## Features
- Wireless Bluetooth control
- Fast and powerful movement
- Compact and lightweight design
- Easy maneuverability
- Rechargeable power supply

## Components Used
- Arduino UNO / ESP32
- HC-05 Bluetooth Module
- Motor Driver Module (L298N)
- DC Motors
- Robot Chassis
- Wheels
- Battery
- Jumper Wires

## Working Principle
The Bluetooth module receives commands from a mobile phone application and sends them to the microcontroller. The microcontroller processes the commands and controls the motor driver module, which operates the DC motors to move the robot forward, backward, left, or right during the sumo match.

## Applications
- Robotics competitions
- Educational robotics projects
- Wireless robot control systems
- Embedded systems learning

## Advantages
- Easy wireless operation
- Simple circuit design
- Low-cost implementation
- Improves robotics programming skills

## Future Scope
The robot can be upgraded with autonomous navigation, obstacle detection sensors, AI-based movement strategies, and improved motor power for advanced robotics competitions.
